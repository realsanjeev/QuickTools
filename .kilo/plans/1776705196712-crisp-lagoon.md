# QuickTools - Multi-Utility Web Application

## Project Overview
- **Type**: Single-page web application (React)
- **Core functionality**: Host multiple utility tools with QR code generator and Unicode text formatter as initial tools
- **Target users**: General users needing quick utility tools

## UI/UX Specification

### Visual Design (based on bhandarisanjeev.com.np)
- **Theme**: Clean, modern, minimalist with dark mode aesthetic
- **Primary color**: Dark backgrounds (#0a0a0a, #1a1a1a) with accent colors
- **Typography**: Modern sans-serif (Inter, system-ui)
- **Layout**: 
  - Fixed sidebar navigation (left)
  - Main content area (right)
  - Mobile: hamburger menu with slide-out drawer

### Layout Structure
```
+----------------------------------+
|  Logo                            |
+----------+-----------------------+
|          |                       |
|  Tool    |   Tool Content        |
|  Nav     |   (Dynamic)           |
|          |                       |
|          |                       |
+----------+-----------------------+
```

### Pages/Views
1. **Home** - Welcome/landing with tool cards
2. **QR Code Generator** - Tool page
3. **Text Formatter** - Tool page

## Functionality Specification

### Tool 1: QR Code Generator
- **Input types**: URL, Plain Text, Email, SMS, WiFi, vCard
- **Customization options**:
  - Foreground/background color picker
  - Size selection (128px - 1024px)
  - Error correction level (L, M, Q, H)
- **Output**: 
  - Live preview
  - Download as PNG/SVG

### Tool 2: Unicode Text Formatter
- **Styles to implement**:
  - Bold (serif & sans)
  - Italic (serif & sans)
  - Bold + Italic
  - Monospace
  - Cursive/Script
  - Strikethrough
  - Underline
  - Bubble text
  - Square text
  - Small caps
  - Superscript/Subscript
- **Features**:
  - Live preview
  - One-click copy to clipboard
  - Character count display

### Architecture
- **Framework**: React 18+ with Vite
- **Routing**: React Router v6
- **Styling**: CSS Modules or Tailwind CSS
- **QR Library**: qrcode.react or qrcode
- **State**: React Context for global state

## Acceptance Criteria
1. App loads without errors
2. Navigation between tools works smoothly
3. QR codes generate correctly for all input types
4. Text formatting produces correct Unicode output
5. Copy to clipboard works for all text styles
6. Responsive design works on mobile/tablet/desktop
7. Download functionality works for QR codes

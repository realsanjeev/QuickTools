# QuickTools

QuickTools is a modern, lightweight collection of essential utility tools designed for speed and simplicity. Built with a premium aesthetic and high performance in mind, it provides a seamless experience for everyday digital tasks.

## 🛠️ Current Tools
- **QR Code Generator**: Create customizable, high-quality QR codes for links and text.
- **Social Media Text Formatter**: Convert plain text into bold, italic, and stylish formats for social platforms.

## 🚀 Key Features
- **Modern UI**: Clean, glassmorphic design inspired by premium aesthetic standards.
- **Fast & Lightweight**: Built with Vite and React for near-instant load times.
- **Privacy Focused**: All processing happens locally in your browser.
